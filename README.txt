WERTGARANTIE Schadenzertifikat Web-App

Bereitstellung auf Netlify:
1. Ordner 'wertgarantie_webapp' entpacken.
2. Auf https://app.netlify.com/drop gehen.
3. Den gesamten Ordner in das Drag-&-Drop-Feld ziehen.
4. Netlify erzeugt einen Link mit netlify.app.
5. Diesen Link am Laptop, iPad oder Handy in Safari/Chrome/Edge öffnen.

Wichtig:
- Nicht nur die index.html aus iCloud/Dateien öffnen, sondern den Netlify-Link verwenden.
- Die Web-App ist eine statische Ein-Datei-App. Es werden keine Kundendaten gespeichert oder an einen Server gesendet.
